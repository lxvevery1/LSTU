В РФ программа - это данные, предназначенные для управления конкретными компонентами системы обработки информации в целях реализации определенного алгоритма.
Исходный модуль - программный модуль на исходном языке, обрабатываемый транслятором и представлемый для него как целое, достаточное для проведения трансляции.
Объектный модуль - программный модуль, получаемый в результате трансляции исходного модуля. Программа на машинном языке с неразрешенными внешними ссылками.
Загрузочный модуль - программный модуль, представленный в форме пригодной для загрузки в ОЗУ для выполнения. Сохраняется во внешней памяти в виде файла.
Программное средство - объект, состоящий из программ, процедур, правил, а также сопутствующим им документацией и данных, относящийся к функционированию системы
обработки информации.
Программное обеспечение - совокупность программ, систем и обработки информации и программных документов, необходимых для их эксплуатации с шагом.
Алгоритм - конечный набор заданных инструкций для выполнения определенной задачи, результатом которого при заданном исходном состоянии будет соответствующая
распознаваемое конечное состояние
Компьютерная программа - последовательность инструкций, которую компьютер способен интерпретировать и выполнить.
Исходный код - любые последовательности выражений, записанные на одном из человекочитаемых языков программирования компьютеров.
Компьютерное информационное обеспечение - одна или большее число компьютерных программ и блоков данных, хранимых на накопителе компьютера для определенных целей.
Программное обеспечение выполняет функцию согласно программе, которую оно реализует либо прямо задавая инструкцию аппаратному обеспечению компьютера, либо обеспечивая
входными данными другие элементы информационного обеспечения. Данные существуюи исключительно для их использования программным обеспечением.
Алгоритм - конечный упорядоченный набор правил для решения проблемы.
программа - Компьютерная программа.
Синтаксический блок - соответствующие правилу конкретного языка программирования и состоящий из объявлений и утверждений или инструкций необходимых для выполнения
определенных функций решения задач или проблем.
Прикладная программа - программное средство или программа, специально предназначенная для решения прикладных проблем определенного класса.
Пакет программных средств - программный пакет. Полный и документированный набор программ предназначенный для того чтобы многие пользователи создавали свои приложения
Программные обеспечения - любая часть программ, процедур, правил и документация для систем обработки информации.
Программа для ЭВМ - объективная форма представления совокупности данных и команд, предназначенных для функционирования ЭВМ и других компьютерных устройств с целью
получения определенного результата. Под программной для ЭВМ подразумеваются также подготовительные материалы, полученные в ходе её разработки и пораждаемые ею
аудио-визуальные отображения.
База данных - это объективная форма представления и организации совокупности данных систематизированных таким образом, что эти данные могли быть найдены и обработаны
с помощью ЭВМ.Понятие "Базы данных" != техническому понятию, обозначающему именно данные, упорядоченные определенным образом, а является аналогом понятия "структура бд"
Компьютерная программа - набор выражений или инструкций для прямого или косвенного использования в компьютере с целью достичь определенного результата.
Коннигиляция - в том числе база данных - работа сформированная посредством сбора и увязки существовавших ранее материалов или данных, которые подобраны, скомпонованы
или упорядочены таким образом, что результат является оригинальной авторской работой. Программный продукт - как объект права - декомпозиция обозначающая отдельный
продукт прямым выделением следующих структурных составляющих:
    1. Технология автоматизированной обработки данных, заложенная в программный продукт.
    2. Математические методы и модели, используемые в рамках технологии.
    3. Набор алгоритмов, реализующие технологию для данной компьютерной архитектуры.
    4. Ноу-хау и секреты производства обеспечивающие умпешную реализацию технологий.
    5. Исходный код программного продукта, реализующий алгоритм на одном из языке программирования.
    6. Скомпилированный, исполняемый двичный код программного продукта для данной программно-аппаратной платформы
    7. Элементы дизайна художественные, литературные и аудиовизуальные произведения, являющиеся частью пользовательского интерфейса программного продукта.
    8. Техническая и пользовательская документация
    9. Средства индивидуализации программного продукта.
