Коэффициенты дробные, АГА. Значит можно применить метод Гомори

**Метод Гомори.**  
В полученном оптимальном плане присутствуют дробные числа.  
По 2-у уравнению с переменной x1, получившей нецелочисленное значение в оптимальном плане с наибольшей дробной частью 11/18, составляем дополнительное ограничение:  
q2 - q21•x1 - q22•x2 - q23•x3 - q24•x4 - q25•x5 - q26•x6≤0  
q2 = b2 - [b2] = 65/18 - 3 = 11/18  
q24 = a24 - [a24] = -1/18 + 1 = 17/18  
q26 = a26 - [a26] = 1/9 - 0 = 1/9  
Дополнительное ограничение имеет вид:  
11/18-17/18x4-1/9x6 ≤ 0  
Преобразуем полученное неравенство в уравнение:  
11/18-17/18x4-1/9x6 + x7 = 0  
коэффициенты которого введем дополнительной строкой в оптимальную симплексную таблицу.  
Поскольку двойственный симплекс-метод используется для поиска минимума целевой функции, делаем преобразование F(x) = -F(X).
