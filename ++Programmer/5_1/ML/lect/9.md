Нелинейная - матрица Якоби.
Каждый шаг решаем линейную задачу, приближаясь
к решению нелинейной задачи.

Применение методов локальной оптимизации ограниченно. Находит решение в окрестностях
некоторой стартовой точки.
Ищем глобальный экстремум.
Метод мультистартер - в начале несколько стартовых точек.
В резульатате подбираем лучший старт.

L1 - задача наименьших модулей.
Если сравнивать решение задачи L1 и она не всегда дифференцируема.
Если используется линейная по параметрам модель, то норма более робастна.
Для линейного случая L2 имеет единственное решение. Для L1 множество.

MAE - медианная регрессия (частный случай квантильной регрессии)
Квантиль уровня a, вероятность того что значение случайной величины xa <= a.
Выделяются первый, второй и третий квартили.

Когда решаем задачу квантильной регрессии, то получается следующий минимизируемый
функционал контроля качества.

Квантильная регрессия - задача линейного программирования.
Минимизируемая функция должна быть линейной по параметрам.
Значения всех переменных >= 0.

MAPE - взвешенная медианная регрессия.
Задача решается с использованием квантильной регрессии.
Норма MAPE, если целью работы является минимизация MAPE, то нужно применять
специализированные методы минимизации для MAPE.
Взвешенная => весы = 1 / (|~y_i|)
