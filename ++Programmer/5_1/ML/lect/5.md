- взвешенная сумма.
Это хорошо для задачи регрессии. Логистическая регрессия употребляется когда есть 2 класса.
Для того чтобы определить к какому классу относится объект с вектором X. Задаем значение
для (тетта), тогда входной вектор X будет принадлежать классу C1, а если меньше, то
считаем что объект будет принадлежать классу C0.

Перцептрон - тоже ведется сравнение с пороговым значением тетта.
Задача должна быть линейно разделимой - провести прямую, разделяющую на 2 пространства.
В данном случае необязательно задача должна быть линейно разделимой.

Очень часто в качестве порогового значения выбирается [0; 1] ~ 0.5.

Загадка - ответ да или нет - четкая классификация.
Тот кто пытается узнать загаданное понятие, если использует не более 20 вопросов -
он это угадает.
делим на 2 части - 2^20=1048576 понятий. Деревья решений применяются для решения задач
регрессии.

Строится бинарное(почти всегда) дерево.
Каждая из дуг помечается ответом T/F.
Если значение какого-то признака больше, дто идем по одной вектке,
если нет, то по другой.
