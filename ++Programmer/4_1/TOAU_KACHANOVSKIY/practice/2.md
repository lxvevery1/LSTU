Как минимизировать?

На фирме 3 монитора, 2 клавиатуры

{
    a11x1+a12x2+a13x3<=T1
    a21x1+a22x2+a23x3<=T2
    x1,x2,x3>=0
}

x1 - кол-во узлов первого вида
x_1_1 - первый индекс завод, а авторой производительность

Нужно иметь 6 переменных
сколько узлов каждого типа -

штуки/штуки_в_час

    x11/a11 + x12/a12 + x13/a13 <= T1  // x11/a11=время для производства первого узла первым заводом

## Подзадача
Планирование производства
Предприятие проивзодит N видов продукции, причем затраты на производство продукции j из постоянных затрат, не зависит от продукции
и переменные затраты c - зависят от объема продукции
з_j = k_j + c_j * x_j

Есть затраты, которые пропорциональны объему.

Для производства продукции m-видов сырья
Для производства единицы продукции a_ij - расход i-го ресурса (сырья) на 1ед j-той продукции
d_j - реализация - не больше!
p_j - цена
b_i - запас i-го ресурса

Требуется определить объем каждого
Прибыль_j = Выручка_j - Затраты_j

Если продукция производится, то з_j = k_j + c_j * x_j
а если нет, то и постоянные затраты тоже должны уйти

Затраты при x=0 тоже должны быть=0, а не k_j
